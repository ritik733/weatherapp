#Introduction
This is a simple weather forecast application built using Tkinter in Python. The application allows users to select a city from a list and retrieve real-time weather information using the OpenWeatherMap API.

# Features
City Selection: Users can choose a city from a predefined list.
Weather Information: The application displays the current weather, weather description, temperature, and pressure.
Temperature Conversion: The temperature is displayed in Celsius.
